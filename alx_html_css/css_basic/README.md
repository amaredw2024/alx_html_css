# CSS Basics Project

This project introduces the fundamentals of styling webpages using CSS.  
It is part of the ALX Front-End Curriculum.

## Project Requirements

- All files must end with a new line.
- A `README.md` file is required at the root of this project folder.
- No external libraries or frameworks are allowed.
  - **Only HTML, CSS, and JavaScript** may be used.
  - No NodeJS, React, Bootstrap, VueJS, etc.
- All code must be **W3C compliant** and validated with the **W3C Validator**.
- You must create a new directory called `css_basic` inside the repository `alx_html_css`.

## Project Files

Inside the `css_basic` directory:

- `index.html` – copied from `html_basic`
- `tweets.html` – copied from `html_basic`
- `base.css` – contains the base styling rules
- `styles.css` – empty CSS file where you will write your custom styles

## Adding CSS Files to HTML

Inside each HTML file (`index.html` and `tweets.html`), add the following lines inside the `<head>` tag:

```html
<link href="base.css" rel="stylesheet">
<link href="styles.css" rel="stylesheet">
